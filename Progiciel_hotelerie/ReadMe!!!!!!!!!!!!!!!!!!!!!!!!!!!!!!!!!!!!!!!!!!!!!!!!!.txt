Merci de bien suivre les étapes d'installation présentes dans ce fichier pour le bon fonctionnement du projet : 

1 - Installer le jdk Java en version 1.8.0_121 pour éviter tout type de conflit de version 
		http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

2 - Télécharger et installer l'éditeur graphique Scene Builder pour créer les vues JavaFX
		http://javafx-scene-builder.software.informer.com/download/

3 - Importer la librairie JFoenix dans Scene Builder.

4 - Lier Scene Builder à votre IDE
		- Dans eclipse, clique droit sur un fichier FXML
		- Open With -> Other -> External program -> Ciblez l'exécutable du Scene Builder
		- Cochez les 2 cases qui attribuent l'éditeur par défaut à ces fichiers

5 - Enfin, importez toutes les librairies qui sont dans le dossier Librairie situé à la racine du projet.

6 - Vous pouvez coder !!!!!!!!!!!!
